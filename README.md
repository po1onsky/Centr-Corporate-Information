# Telegram Mini Store

Минимальный каркас Telegram Mini App с витриной онлайн-магазина, выбором пола, категориями и блоком популярных товаров.

## Возможности
- Переключение между мужскими и женскими коллекциями.
- Категории внутри каждого раздела.
- Главный экран с популярными товарами.
- Лёгкий API и база SQLite для фиксации посещений.

## Запуск

```bash
npm install
npm run start
```

Приложение будет доступно по адресу `http://localhost:3000`.

## API
- `GET /api/popular` — список популярных товаров.
- `GET /api/categories?gender=male|female` — категории для выбранного пола.
- `GET /api/products?gender=male|female&category=...` — товары по фильтру.
- `POST /api/track` — запись посещения.
- `GET /api/visits` — последние 50 записей посещений.
